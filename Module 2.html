Object-Oriented Information Systems

Learning Outcomes

  • Information Systems and their Importance.
    • Examine and evaluate core elements of information systems and their importance.
    • Examine SDLC phases and common issues that can arise when deploying a system.
  • Fundamentals of Object-Oriented Design, Development and Python.
    • Identify and develop object-oriented design for a system.
    • Design object-oriented models and develop object-oriented software using Python.
  • Understanding and working UML.
    • Use UML to develop and document an object-oriented system design.
  • Database Design and Implementing Database with SQL.
    • Develop database design principles and design correctly normalised database
    • Design and implement databases and table structures using SQL and use ERD to document design.
    • Develop queries using SQL and examine access to data and security issues around SQL.
  • Web Development in Python.
    • Design and implement an object-oriented information system using Flask, Python and SQL.
  • The Future of Information Systems.
    • Examin and critically evaluate emerging technologies on privacy in information systems.
